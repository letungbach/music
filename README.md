# Solfège Exercise Number 1 - F Major

An interactive web-based music education tool for practicing solfège (sight-singing) in the key of F Major. This exercise features a traditional Vietnamese-style interface with parchment aesthetics and multiple instrument options for auditory learning.

## Features

- **Visual Score Display**: Professional music notation rendered using VexFlow, showing two staves of F Major solfège exercises
- **Interactive Playback**: Play the exercise with different instrument timbres to aid in pitch recognition
- **Instrument Selection**: Choose from Piano, Violin, Guitar, Cello, or Vocal synthesis
- **Solfège Annotations**: Each note displays its corresponding solfège syllable (Fa, Sol, La, Ti, Do, Rê)
- **Responsive Design**: Clean, parchment-themed interface with smooth animations

## How to Use

1. Open `index.html` in a modern web browser
2. Select your preferred instrument from the options below the score
3. Click "Phát nhạc" (Play Music) to start playback
4. Listen to the melody and follow along with the visual highlighting
5. Use "Dừng" (Stop) to halt playback at any time

## Technologies Used

- **VexFlow**: For rendering musical notation
- **Tone.js**: For audio synthesis and instrument simulation
- **HTML5/CSS3**: For structure and styling
- **JavaScript**: For interactivity and playback control

## Browser Requirements

- Modern browser with Web Audio API support (Chrome, Firefox, Safari, Edge)
- Audio output enabled
- JavaScript enabled

## Educational Context

This exercise is designed for music students learning to associate solfège syllables with their corresponding pitches in F Major. The sequence includes common melodic patterns and intervals to build sight-singing proficiency.
